# This 3rdEYE Chrome Extension is developed by Team CS

## Installation Guide

Follow the steps below to load the unpacked 3rdEYE Chrome Extension into your browser:

1. **Open Chrome Browser**
   - Launch Google Chrome on your computer.

2. **Access the Extensions Page**
   - In the Chrome address bar, type `chrome://extensions/` and press `Enter`.

3. **Enable Developer Mode**
   - Once on the Extensions page, locate the toggle in the upper right corner labeled `Developer mode`. Switch it on. This will enable developer options for managing extensions.

4. **Load Unpacked Extension**
   - Click the `Load unpacked` button that appears after enabling Developer Mode.
   
5. **Select the Extension Folder**
   - In the pop-up window, navigate to the folder where your extension files are located (this folder should include the manifest.json file). 
   - Select the folder and click `Open`.

6. **Verify Extension Installation**
   - Once the extension is loaded, you should see it listed on the Extensions page. If there are any errors, Chrome will display a notification.
   - Ensure the extension is activated by checking the switch next to the extension name.

You have now successfully installed the unpacked Chrome extension!
